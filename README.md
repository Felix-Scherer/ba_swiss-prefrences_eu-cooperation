# ba_swiss-prefrences_eu-cooperation
Bachelor Thesis: Societal Preferences and Switzerland-EU Cooperation (2010-2024)

Diese Arbeit untersucht den Zusammenhang zwischen gesellschaftlichen Präferenzen der Schweizer Bevölkerung und der institutionellen Verflechtung mit der Europäischen Union im Zeitraum 2010–2024. Basierend auf dem Neuen Liberalismus wird argumentiert, dass Kooperationsbereitschaft und Autonomiepräferenzen die Aussenpolitik massgeblich beeinflussen. Ein selbst konstruierter EU-Verflechtungsindex (PCA) dient als abhängige Variable. Die Ergebnisse zeigen signifikante Zusammenhänge zwischen Präferenzdimensionen und EU-Verflechtung (R² = 0.95).
